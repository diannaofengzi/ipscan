# ipscan

一个支持 IPv4 和 IPv6 的端口扫描程序，专注于扫描指定端口并识别服务类型。

## 功能特点

- 🌐 **双栈支持**：同时支持 IPv4 和 IPv6 地址扫描
- 🎯 **目标端口**：默认扫描 19890 和 18789 端口（可自定义）
- 🔍 **服务识别**：自动识别端口服务类型并获取服务横幅
- ⚡ **高性能**：多线程并发扫描，可自定义线程数
- 📊 **实时显示**：发现开放端口立即显示详细信息
- 💾 **报告导出**：支持将扫描结果保存到文件
- 🛑 **优雅退出**：支持 Ctrl+C 随时终止扫描

## 安装要求

- Python 3.6+
- 无需额外依赖（仅使用标准库）

## 使用方法

### 基本用法

```bash
# 扫描单个 IPv4 地址
python3 portscan.py -i 192.168.1.1

# 扫描单个 IPv6 地址
python3 portscan.py -i 2001:db8::1

# 扫描 IPv4 网段
python3 portscan.py -i 192.168.1.0/24

# 扫描 IPv6 网段
python3 portscan.py -i 2001:db8::/64

# 混合扫描 IPv4 和 IPv6
python3 portscan.py -i 192.168.1.0/24 2001:db8::/64
```

### 从文件读取 IP 列表

创建 IP 列表文件（每行一个 IP 或网段，`#` 开头为注释）：

```text
# IPv4 地址
192.168.1.0/24
10.0.0.1

# IPv6 地址
2001:db8::/64
fe80::1
```

然后执行：

```bash
python3 portscan.py -f ip_list.txt
```

### 高级选项

```bash
# 自定义线程数和超时时间
python3 portscan.py -i 10.0.0.0/8 -t 100 -T 3.0

# 自定义扫描端口
python3 portscan.py -i 192.168.1.0/24 --ports 80 443 8080

# 输出报告到文件
python3 portscan.py -i 192.168.1.0/24 -o report.txt

# 禁用实时显示（仅在结束时显示报告）
python3 portscan.py -i 192.168.1.0/24 --no-realtime

# 显示详细进度
python3 portscan.py -i 192.168.1.0/24 -v
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-i`, `--ip` | IP 地址或网段（CIDR 格式），可多个 | - |
| `-f`, `--file` | 包含 IP 地址列表的文件路径 | - |
| `-t`, `--threads` | 并发线程数 | 50 |
| `-T`, `--timeout` | 连接超时时间（秒） | 2.0 |
| `-o`, `--output` | 输出报告文件路径 | - |
| `-v`, `--verbose` | 显示详细扫描进度 | False |
| `--no-realtime` | 禁用实时显示 | False |
| `--ports` | 自定义扫描端口 | [19890, 18789] |

## 注意事项

1. **法律合规**：请确保您有权扫描目标网络，未经授权扫描他人网络可能违法
2. **网络影响**：大量并发扫描可能影响网络性能，建议适当调整线程数
3. **防火墙**：某些端口可能被防火墙阻止，导致扫描结果不准确
4. **IPv6 支持**：确保系统已启用 IPv6 才能扫描 IPv6 地址
